Generalized Dual numbers
=========================

.. py:currentmodule:: pyaudi

The following instances of generalized dual numbers (and thus the corresponding algebra) are made available 
in the python module pyaudi:

 * :py:class:`gdual_double`: basic type for computing on real numbers
 * :py:class:`gdual_vdouble`: basic type for computing on vectors of real numbers
